## 📌 Descripción

**avance_topicos** es una aplicación móvil desarrollada con **Flutter** que funciona como un **sistema de punto de venta (POS)**, permitiendo gestionar ventas, productos y clientes desde un dispositivo móvil.

Está diseñada para ofrecer una solución rápida, moderna y portátil para negocios pequeños y medianos.

---

## 🚀 Características

- 🛒 Registro de ventas desde el móvil  
- 📦 Gestión de productos  
- 👥 Administración de clientes  
- 💵 Control de ingresos  
- 📱 Interfaz intuitiva y responsive  
- ⚡ Alto rendimiento multiplataforma
