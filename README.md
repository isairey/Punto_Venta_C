# 🌌 Acerca de POS Móvil Flutter

**POS Móvil Flutter** es una aplicación móvil desarrollada para funcionar como un sistema de punto de venta portátil, permitiendo administrar ventas, productos y clientes directamente desde un smartphone o tablet.

La aplicación está enfocada en ofrecer movilidad, rapidez y una experiencia moderna para pequeños y medianos negocios.

La plataforma permite:

- 🛒 Registro de ventas móviles
- 📦 Gestión de productos
- 👥 Administración de clientes
- 💰 Control de ingresos
- 📊 Historial de ventas
- 📱 Interfaz responsive
- ⚡ Alto rendimiento
- 🌐 Integración API REST

El proyecto fue desarrollado para practicar:

- Flutter
- Dart
- Desarrollo móvil
- Arquitectura móvil
- Sistemas POS
- APIs REST
- Gestión empresarial

# ⚡ Instalación

---


---

## 3️⃣ Instalar dependencias

```bash

```
